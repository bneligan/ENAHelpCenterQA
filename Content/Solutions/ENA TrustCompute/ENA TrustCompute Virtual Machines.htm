ENA TrustCompute Virtual Machines

You view, sort, and search for existing VMs here as well as view both Overview and Performance data. You can also click New VM to open the Create New Virtual Machine.

You select the zone for which you would like to view Virtual Machines from the Zone dropdown menu in the top right.

Overview tab

Displays basic and capacity information about your VMs.

Performance tab

Displays basic and usage information about your VMs.

If a VM is stopped, all of its counters will be 0.

Apply Filters

You can apply filters to the Virtual Machine information in both the Overview and Performance tab.

Show by Group

When the Show by Group slider is toggled On, VMs are sorted under group names. Click anywhere in the group header bar to open the dropdown showing the VMs in a group. To go back to viewing all VMs, toggle the slider to Off.

Status

To see only a list of VMs in a particular state, select a state from the Status dropdown menu.

Find Entity

To search for a VM by name type the name or part of the name in the Find Entity box and click GO. To clear text from the Find Entity box, click Clear or delete the entered text and click GO.

New VM

Opens the Create New Virtual Machine screen.

Sorting VMs by Column Heading

You can sort the list of VMs by the data columns by clicking on the column header in both the Overview and Performance tabs. This is true of the list of all VMs and for a list of VMs in a group when Show by Group is toggled on.

Viewing Details for a VM

When you click on a VM name, you see basic capacity and usage data, as well as management controls.

You can click on Full Detailsto see more detail as well as a VM History graph

On the VM History graph, you can filter your data by:

  1. Selecting type of data you would like to Show on Y Axis from the dropdown menu
  2. Selecting the Time Interval for which you would like to view data from the dropdown menu
  3. Clicking on the red or blue boxes to toggle read and/or write data on or off on graph

Stop a VM

Click Stop on a details screen and click Stop again in the confirmation dialogue box to stop a VM. You can still view the full details of a VM that is in a Stopping state, but all other options are unavailable until it is fully stopped. A fully stopped VM can be modified, deleted, reinstalled, or have its disks customized, but the console is unavailable. To run a stopped VM, click Run.

Delete a VM

Deletes the VM and sets its status to Destroyed, where it will remain for 24 hours before being automatically deleted permanently. A destroyed VM can either be deleted permanently or restored within 24 hours. If a VM is deleted permanently, it is erased and unrecoverable. If a VM is restored, it reverts to its state prior to being deleted.

VMs are restored to a Stopped state. Click Run to restart.

When a VM is permanently deleted, anyone in your organization working in ENA TrustCompute receives a notice to refresh the VM list. Refreshing removes the deleted VM from the list.
