ENA Bandwidth Utilization

You use the ENA Bandwidth Utilization tool to track and monitor aggregate bandwidth usage by site using industry standard metrics. This information assists in troubleshooting, planning future capacity requirements, and tracking usage spikes.

You log into the ENA Bandwidth Utilization tool from my.ena.com

If you have questions about or would like support in using the ENA Bandwidth Utilization tool, please contact the ENA CTAC at support@ena.com or (888) 612-2880.
