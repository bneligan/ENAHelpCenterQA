ENA Network Availability Monitoring

The ENA Network Availability Monitoring tool displays real-time status of a site's network, allowing insight into the health of the network at any time, from any place with an Internet connection.

You log into the ENA Network Availability Monitoring tool from my.ena.com

If you have questions about or would like support in using the ENA Network Availability Monitoring tool, please contact the ENA CTAC at support@ena.com or (888) 612-2880.
